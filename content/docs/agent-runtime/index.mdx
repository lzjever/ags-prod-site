# 智能体运行专题

这一组内容用来回答一个最关键的问题: AgentSmith 为什么不只是提供一个 Agent 调用入口，而是在尝试把智能体执行升级成团队可以长期采用的正式能力。

如果你更关心产品差异化，建议先从“为什么不只是一个 Agent UI”开始；如果你更关心任务是如何真正跑起来的，再继续看运行链路页。两页合起来，基本能把这条主线讲清楚。

  eyebrow="Topic"
