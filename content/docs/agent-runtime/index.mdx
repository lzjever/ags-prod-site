# 智能体运行专题

这一组内容展示 AgentSmith 如何围绕任务、文件、智能体和运行结果组织完整执行链路，让智能体执行能够在项目中持续保留、持续复用并持续协作。

<ValueStatement
  eyebrow="Runtime"
  title="智能体执行会在项目中形成持续可用的工作链路。"
  summary="任务如何创建、输入如何组织、过程如何回传、结果如何沉淀，都会在同一套项目结构中被保留下来。"
  detail="因此，执行结果不只停留在单次调用中，而可以继续进入后续协作和治理过程。"
