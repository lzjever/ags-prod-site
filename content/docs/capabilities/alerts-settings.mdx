# Alerts 与 Settings

Alerts 与 Settings 在站点中不作为主叙事中心，但它们分别承担治理信号与项目配置维护职责。

<SiteSection title="Alerts" description="Alerts 是治理信号，不是发布流水线或告警编排系统。">

1. 承接异常热点与治理提醒。
2. 与可追溯证据关联。
3. 为空时不制造噪音。

</SiteSection>

<SiteSection title="Settings" description="Settings 用于维护项目基础信息和治理边界，是项目生命周期管理的重要入口。">

1. 维护项目基础配置。
2. 承接治理边界相关设置。
3. 不应被误解为另一个复杂后台中心。

</SiteSection>
