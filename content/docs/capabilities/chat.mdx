# Chat

Chat 是项目内对话式 AI 使用面，负责解决“快速开始”和“低门槛使用”的问题。

<SiteSection title="产品定位" description="它应该是用户进入项目后最自然的 AI 入口。">

1. 基于 endpoint 或 external agent 发起对话。
2. 承接项目上下文中的快速分析、问答与迭代。
3. 保持结果优先，不把底层执行协议直接暴露给普通用户。

</SiteSection>

<SiteSection title="为什么重要" description="Chat 不是独立治理体系，而是 endpoint 统一约束链路的一部分。">

1. 发送前必须绑定可用执行对象。
2. 对附件、多模态能力和错误映射保持前置判断。
3. 与 Notebook、API 路径共享统一治理逻辑。

</SiteSection>
