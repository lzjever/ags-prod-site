# Credentials

Credentials 模块用于集中管理对接第三方模型和服务所需的项目级凭据。

<SiteSection title="产品意义" description="它帮助把散落的连接配置变成项目级治理对象。">

1. 集中管理凭据。
2. 与 endpoints 和 agents 形成明确关联。
3. 降低凭据散落带来的安全与维护成本。

</SiteSection>

<SiteSection title="当前边界" description="当前 secret handling 仍然是 MVP 级姿态，站点表达不能把它包装成最终安全态。">

1. 页面与交互已实现。
2. 权限门禁已接入。
3. 更强的 secret 安全姿态仍是后续演进方向。

</SiteSection>
