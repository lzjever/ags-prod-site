# Credentials

Credentials 模块用于集中管理对接第三方模型和服务所需的项目级凭据。

<SiteSection title="产品意义" description="它帮助把散落的连接配置变成项目级治理对象。">

1. 集中管理凭据。
2. 与 endpoints 和 agents 形成明确关联。
3. 降低凭据散落带来的安全与维护成本。

</SiteSection>

<SiteSection title="当前阶段" description="当前 secret handling 已覆盖主要交互与权限门禁，但仍会继续演进。">

1. 页面与交互已实现。
2. 权限门禁已接入。
3. 更强的 secret 安全姿态仍是后续演进方向。

</SiteSection>

<SiteSection title="典型使用方式" description="Credentials 本身不是终点，而是支撑模型接入和智能体执行的基础能力。">

1. 管理员先录入项目级凭据。
2. Endpoint 创建时绑定相应凭据。
3. Chat、Notebook 与 Agents 在统一治理条件下复用这些接入能力。

</SiteSection>
