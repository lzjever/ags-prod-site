# Endpoints

Endpoints 是当前项目级治理主线的核心资源对象。

<SiteSection
  title="需求定位"
  description="后续 Chat、Notebook 与 API 路径都围绕同一套 endpoint 资源工作。"
>

1. 统一管理项目可用模型资源。
2. 让能力边界、资源来源与成本边界集中配置。
3. 为 Usage、Policy、Audit 提供共同主对象。

</SiteSection>

<SiteSection title="关键特征" description="Endpoint 是治理主线中的核心资源对象，而不只是普通配置列表。">

1. endpoint 列表与 CRUD。
2. capability-first 的 provider / model 选择。
3. import / export。
4. Chat 仅展示 completion-capable endpoint。

</SiteSection>
