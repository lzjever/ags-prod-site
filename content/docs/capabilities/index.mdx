# 核心能力总览

AgentSmith 的能力结构不是“很多模块的集合”，而是围绕项目级 AI 使用、治理与智能体运行形成统一能力骨架。

<ValueStatement
  eyebrow="Capabilities"
  title="每一项能力都会继续连接到同一个项目工作流。"
  summary="Chat 负责降低起步门槛，Notebook 负责承接复杂执行，Files 负责沉淀项目资产，Agents 负责连接执行能力，Governance 则确保这一切可以被长期理解和持续管理。"
  detail="因此，团队在日常使用、任务执行、文件管理和资源治理之间不需要重新切换工作方式。"
/>

<Cards>
  <Card title="Chat" href="/docs/capabilities/chat" description="让团队先在项目里用起来，再逐步进入更完整的执行与治理链路。" />
  <Card title="Notebook" href="/docs/capabilities/notebook" description="把复杂执行升级成可回看、可复用、可协作的长期任务。" />
  <Card title="Files" href="/docs/capabilities/files" description="把输入资料和运行产物沉淀成项目资产，而不是一次性附件。" />
  <Card title="Agents" href="/docs/capabilities/agents" description="把原本依赖个人环境的执行能力纳入团队正式工作流。" />
  <Card title="Governance" href="/docs/capabilities/governance" description="让资源边界、消耗观察和审计回看围绕同一主线持续生效。" />
</Cards>
