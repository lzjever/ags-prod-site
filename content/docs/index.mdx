# AgentSmith 导览

如果你第一次接触 AgentSmith，可以先把它理解成一套帮助团队长期使用 AI 的产品系统。它既负责组织工作区、项目、资源、成员和审计，也负责承接对话、任务、文件和智能体执行。对用户来说，这意味着日常使用和后续管理不必被拆成两套彼此脱节的流程。

很多团队第一次接入 AI 时，最先遇到的并不是模型问题，而是工作如何继续推进。对话之后怎么进入正式任务，任务结果放在哪里，谁能继续接手，资源限制如何生效，异常发生后去哪里回看。这些问题如果长期分散在不同工具里，AI 就很难真正进入业务。

这组内容不是单纯的目录，而是整套产品的阅读入口。你可以先判断这是不是你要找的产品，也可以直接选择最关心的一层，从价值、结构、能力和界面四个角度进入。
