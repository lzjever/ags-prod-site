# 智能体连接信息

<SiteSection title="界面概览" description="此对话框用于查看连接地址和服务密钥等接入信息。">

这个对话框用于查看智能体接入所需的连接地址、服务密钥和其他必要信息。用户可以据此在本地 runner 或对应运行环境中完成连接，并确认该智能体已经准备好进入项目。

</SiteSection>

<SiteSection title="主要价值" description="连接信息对话框把外部执行能力所需的关键信息集中呈现。">

这个对话框把接入所需的信息集中整理在一处，减少了配置过程中的查找和误用风险。连接动作完成后，智能体就可以作为项目中的可用能力被持续使用和管理。

</SiteSection>
