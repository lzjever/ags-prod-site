<SiteSection title="界面概览" description="此对话框用于创建新的智能体，并设置基础属性。">

创建智能体对话框用于定义一个新的项目级智能体。名称、模式、说明和交互方式都会在这里完成设置，为后续连接、调用和管理建立统一基础。

</SiteSection>

<SiteSection title="主要价值" description="创建智能体对话框让执行能力从一开始就具备清晰定义。">

这个对话框让智能体在进入项目之前就具备明确的身份和用途。团队成员后续看到的会是已经定义清楚、可以稳定调用的项目对象。

</SiteSection>
