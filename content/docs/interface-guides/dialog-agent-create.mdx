# 创建智能体对话框

<SiteSection title="你会在这里做什么" description="这里用于创建新的智能体对象，并为后续连接信息与任务执行做准备。">

1. 填写智能体名称、模式和说明。
2. 选择交互方式与默认执行偏好。
3. 保存后继续配置连接信息或服务密钥。

</SiteSection>

<SiteSection title="关键看点" description="创建智能体对话框最适合说明项目如何把智能体能力纳入正式管理。">

1. 智能体名称、模式和交互方式在创建阶段被清晰定义。
2. 这让执行主体不再依赖个人脚本约定，而是进入项目治理范围。
3. 页面很好地体现了“先接入，再使用”的产品节奏。

</SiteSection>
