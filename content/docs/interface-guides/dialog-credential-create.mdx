# 创建凭据对话框

<SiteSection title="界面概览" description="此对话框用于新增项目凭据，并为后续资源接入做准备。">

这个对话框用于录入项目所需的凭据，并为后续模型或第三方服务接入做好准备。凭据创建完成后，可以在凭据管理页持续维护，并在 Endpoint 或其他配置中复用。

</SiteSection>

<SiteSection title="主要价值" description="创建凭据对话框帮助用户以受控方式录入和维护敏感接入信息。">

这个对话框让敏感接入信息能够以统一方式被录入、维护和复用。这样既方便后续资源接入，也能减少凭据散落在个人环境中的风险。

</SiteSection>
