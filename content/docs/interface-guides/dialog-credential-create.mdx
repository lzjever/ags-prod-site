# 创建凭据对话框

<StatusBadge state="已实现" />

<ScreenshotFrame
  preset="dialog-credential-create-guide"
  caption="创建凭据对话框用于录入模型服务或第三方平台所需密钥，是接入 Endpoint 之前的重要准备。"
/>

<SiteSection title="你会在这里做什么" description="这里用于新增可复用的项目凭据，并为后续模型接入做准备。">

1. 填写凭据名称与密钥内容。
2. 保存后回到凭据列表进行统一管理。
3. 在 Endpoint 配置中复用该凭据。

</SiteSection>

<SiteSection title="关键看点" description="创建凭据对话框最适合说明敏感接入信息如何被纳入统一项目管理。">

1. 凭据创建是模型接入与外部能力连接的前置步骤。
2. 页面强调的是受控录入与复用，而不是原始密钥暴露。
3. 这让后续 Endpoint 与 Agent 配置具备统一基础。

</SiteSection>
