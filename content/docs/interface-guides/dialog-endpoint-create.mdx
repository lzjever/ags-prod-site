# 创建 Endpoint 对话框

<StatusBadge state="已实现" />

<ScreenshotFrame
  preset="dialog-endpoint-create-guide"
  caption="创建 Endpoint 对话框用于定义模型入口、协议与凭据绑定，是统一接入模型资源的关键动作。"

<SiteSection title="你会在这里做什么" description="这里用于新增模型接入方式，并把它纳入项目统一治理链路。">

1. 点击创建 Endpoint。
2. 选择引导式或手动配置方式。
3. 保存后在端点列表中查看并继续治理。

</SiteSection>

<SiteSection title="关键看点" description="创建 Endpoint 对话框最适合说明模型资源接入是如何被正式纳入治理主线的。">

1. 用户在引导式与手动配置之间做选择，体现不同接入复杂度。
2. provider、model、credential 等关键关系在这里被建立。
3. 这一步决定了后续聊天、任务与智能体可使用哪些模型资源。

</SiteSection>
