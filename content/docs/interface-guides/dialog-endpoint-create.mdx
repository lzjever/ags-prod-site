# 创建 Endpoint 对话框

<StatusBadge state="已实现" />

<SiteSection title="界面概览" description="此对话框用于新增模型接入方式并设置关键参数。">

这个对话框用于新增模型接入方式，并设置 provider、model 和 credential 等关键参数。无论选择引导式配置还是手动填写，完成后都可以让对应模型资源进入当前项目的可用范围。

</SiteSection>

<SiteSection title="主要价值" description="创建 Endpoint 对话框让模型资源能够以统一方式进入项目资源体系。">

这个对话框为模型资源建立了统一的接入方式。Endpoint 创建完成后，后续的使用策略、用量观察和审计记录都可以围绕同一个对象展开，资源管理也会更加清晰。

</SiteSection>
