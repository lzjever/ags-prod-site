<SiteSection title="你会在这里做什么" description="这里用于创建新的项目文件库，并开始沉淀长期可复用的文件资产。">

1. 填写文件库名称和说明。
2. 创建后在左侧文件库列表中查看新库。
3. 继续上传文件或执行本地挂载。

</SiteSection>

<SiteSection title="关键看点" description="创建文件库对话框最适合说明项目文件并不是临时上传区，而是长期资产容器。">

1. 文件库创建意味着为某类资料建立独立承载空间。
2. 后续上传、挂载和产物回写都会围绕这个文件库继续展开。
3. 这让文件资产从一开始就具备被复用和被管理的基础。

</SiteSection>
