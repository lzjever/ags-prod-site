<SiteSection title="界面概览" description="此对话框用于创建新的文件库并设置基本信息。">

创建文件库对话框用于为一类资料建立新的存储空间。完成创建后，团队可以继续上传文件、配置本地挂载，并让后续任务产物回流到同一文件库中。

</SiteSection>

<SiteSection title="主要价值" description="创建文件库是整理项目资料和承接后续产物的起点。">

文件库建立之后，后续上传、挂载和产物回写都会围绕同一结构展开，资料整理和结果沉淀也会更加清晰，便于长期维护和复用。

</SiteSection>
