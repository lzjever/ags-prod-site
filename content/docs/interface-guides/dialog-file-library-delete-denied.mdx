# 文件库删除限制

<StatusBadge state="已实现" />

<ScreenshotFrame
  preset="dialog-file-library-delete-denied-guide"
  caption="删除限制对话框用于阻止误删仍被引用的文件库，并要求用户明确确认相关名称。"
/>

<SiteSection title="界面概览" description="此对话框用于说明文件库当前无法删除的原因。">

当用户尝试删除仍被引用或仍有内容的文件库时，系统会明确说明当前无法删除的原因，并提示需要确认的条件。这样可以帮助用户在操作前先了解影响范围。

</SiteSection>

<SiteSection title="主要价值" description="删除限制对话框帮助用户避免误删仍在使用中的文件资产。">

这个对话框能够减少误删仍在使用中的文件资产。对于被任务、成员或其他页面继续引用的资料，这种保护机制可以避免后续工作受到影响。

</SiteSection>
