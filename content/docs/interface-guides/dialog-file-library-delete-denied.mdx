# 文件库删除限制

<StatusBadge state="已实现" />

<ScreenshotFrame
  preset="dialog-file-library-delete-denied-guide"
  caption="删除限制对话框用于阻止误删仍被引用的文件库，并要求用户明确确认相关名称。"

<SiteSection title="你会在这里做什么" description="这里用于确认文件库删除限制，并理解当前为什么不能直接删除。">

1. 核对待删除文件库名称。
2. 了解系统给出的限制条件。
3. 清空文件或解除引用后再执行删除。

</SiteSection>

<SiteSection title="关键看点" description="删除限制对话框最适合说明文件资产管理并不是无保护的轻量动作。">

1. 非空文件库不会被轻易删除，系统要求用户明确确认。
2. 这体现了共享资料和项目资产的保护机制。
3. 页面可以很好地支撑“平台重视资产完整性”的表达。

</SiteSection>
