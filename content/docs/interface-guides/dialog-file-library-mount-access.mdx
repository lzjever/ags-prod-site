<SiteSection title="你会在这里做什么" description="这里用于查看并复制本地挂载所需的信息，让 Web 文件库与本地工作目录协同。">

用户会在这里拿到把项目文件库接入本地目录所需的关键信息，包括 metadata 地址、推荐路径和多平台命令。完成挂载后，本地工作目录与 Web 端文件库就能形成连续协作，不再是彼此割裂的两个世界。

</SiteSection>

<SiteSection title="关键看点" description="挂载信息对话框最适合说明 AgentSmith 如何把 Web 文件库与本地目录工作流接起来。">

这张对话框真正有说服力的地方，在于它证明 Files 不是浏览器里的封闭空间。文件库既能在 Web 中被浏览和引用，也能通过标准化的挂载方式进入本地环境，这让项目资产层真正具备了线上线下连续工作的能力。

</SiteSection>
