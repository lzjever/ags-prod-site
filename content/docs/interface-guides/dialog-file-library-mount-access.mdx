<SiteSection title="你会在这里做什么" description="这里用于查看并复制本地挂载所需的信息，让 Web 文件库与本地工作目录协同。">

1. 查看并复制 metadata URL 和推荐路径。
2. 根据平台选择对应挂载命令。
3. 在本地完成挂载后与 Web 端同步操作。

</SiteSection>

<SiteSection title="关键看点" description="挂载信息对话框最适合说明 AgentSmith 如何把 Web 文件库与本地目录工作流接起来。">

1. 文件库不仅能在浏览器里使用，也能通过挂载进入本地环境。
2. 多平台命令和元数据地址让本地接入动作变得标准化。
3. 这进一步说明 Files 是项目资产层，而不是纯 Web 附件区。

</SiteSection>
