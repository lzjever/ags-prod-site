<SiteSection title="界面概览" description="此对话框用于查看本地挂载所需的地址、路径和命令。">

这个对话框用于查看本地挂载所需的地址、路径和命令。用户可以按照这里提供的信息，把项目文件库连接到本地目录，并继续在熟悉的工作环境中处理文件。

</SiteSection>

<SiteSection title="主要价值" description="挂载信息对话框帮助文件库在浏览器和本地目录之间建立连续协作方式。">

这个对话框把 Web 端文件库和本地目录连接起来。用户既可以在浏览器中浏览和管理资料，也可以在本地工具链中继续处理同一批文件，工作方式会更加连贯。

</SiteSection>
