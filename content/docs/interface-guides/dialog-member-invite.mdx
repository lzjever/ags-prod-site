# 邀请成员对话框

<StatusBadge state="已实现" />

<SiteSection title="界面概览" description="此对话框用于邀请成员加入项目并指定其分组。">

邀请成员对话框用于把新成员加入项目，并在加入时指定分组或角色来源。用户可以在这里完成基本身份信息和访问范围的初始设置。

</SiteSection>

<SiteSection title="主要价值" description="邀请成员对话框让成员从加入之初就具备清晰的访问边界。">

这个对话框让成员从加入项目的第一步开始就具备明确的访问边界。后续的协作、资源使用和审计记录都会建立在清晰的成员信息之上。

</SiteSection>
