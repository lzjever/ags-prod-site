# 创建任务对话框

<StatusBadge state="已实现" />

<SiteSection title="界面概览" description="此对话框用于创建新的任务并选择执行智能体。">

创建任务对话框用于为 Notebook 新建一项任务，并在创建时选择执行智能体。任务创建完成后，后续的执行状态、trace 和输出结果都会围绕这一对象持续记录。

</SiteSection>

<SiteSection title="主要价值" description="创建任务对话框让一次执行从一开始就具备明确目标和后续承接关系。">

这个对话框让一次执行从一开始就拥有清晰目标和后续承接关系。相比临时会话，这种任务化的组织方式更适合持续跟踪、多人协作和结果沉淀。

</SiteSection>
