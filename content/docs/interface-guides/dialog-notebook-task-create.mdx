<SiteSection title="关键看点" description="创建任务对话框最适合用来说明 Notebook 与普通对话入口的区别。">

1. 任务从创建开始就是长期对象，而不是一次性消息。
2. 智能体选择发生在任务创建阶段，说明执行能力是被正式绑定的。
3. 这为 trace、artifacts 和后续复用奠定了结构基础。
