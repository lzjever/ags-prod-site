# 创建项目对话框

<StatusBadge state="已实现" />

<ScreenshotFrame

<SiteSection title="界面概览" description="此对话框用于创建新项目并设置基础属性。">

创建项目对话框用于定义新项目的名称、说明和基础属性。项目创建完成后，成员协作、资源管理和任务执行都会在这一边界内展开。

</SiteSection>

<SiteSection title="主要价值" description="创建项目对话框确定了后续成员协作、资源治理和任务执行的共同边界。">

这个对话框为后续的成员协作、资源治理和任务执行确定了共同边界。项目一旦建立，相关资料、能力和记录都会围绕它持续组织。

</SiteSection>
