# 审计事件详情

<SiteSection title="界面概览" description="审计详情抽屉用于查看单条事件的完整上下文。">

当用户需要查看某一条具体事件时，详情抽屉会把事件摘要、责任归属、metadata 和结果集中展示出来，帮助用户在最短时间内掌握这条记录的完整上下文。

</SiteSection>

<SiteSection title="主要价值" description="审计详情抽屉帮助用户快速判断事件影响并继续处理。">

除了展示记录本身，这个详情视图还能帮助用户判断这条事件影响了哪些对象，并继续进入后续处理路径。在排查问题或回看关键变更时，这种组织方式会更加直接。它让管理员不必只停留在抽象列表里，而能快速把一条事件还原成更完整的上下文。

</SiteSection>
