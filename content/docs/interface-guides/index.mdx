# 关键界面导览

本章节汇集了 AgentSmith 中最重要的工作界面。你可以从这里了解一个项目如何承接日常对话、任务执行、文件管理、模型资源配置和审计回看，也可以更快熟悉不同角色在系统中的主要操作方式。

如果你第一次使用 AgentSmith，建议先查看 `项目总览`、`Chat 多轮会话` 和 `Notebook 任务详情`。这三类页面能够帮助你先建立对核心工作流的理解。之后再浏览文件、智能体、资源管理和审计页面，会更容易把整套系统看成一条连续的工作链，而不是若干分散功能。

<SiteSection
  title="推荐阅读顺序"
  description="如果你希望快速建立完整认识，可以按下面顺序浏览。"
>

先看 `项目总览`、`Chat 多轮会话` 和 `Notebook 任务详情`，了解项目中的常见工作如何展开。接着查看 `文件库管理`、`智能体列表` 以及相关创建页面，可以继续了解资料、执行能力和输出结果如何被组织和复用。

最后进入 `Endpoint 管理`、`资源策略`、`用量与限额`、`审计日志` 等页面，就能看到系统如何在资源控制、消耗观察和变更追溯之间保持一致。

</SiteSection>
