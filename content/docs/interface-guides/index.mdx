# 界面与功能说明

这一组页面展示的是 AgentSmith 中最重要的工作界面。你会看到一个项目如何承接日常对话、长期任务、文件资产、智能体接入、模型资源管理和审计回看，也会看到这些页面为什么并不是彼此独立的后台，而是一条连续工作流的不同环节。

如果你想先建立对产品的整体直觉，最值得先看的通常是 `项目总览`、`Chat 多轮会话`、`Notebook 任务详情`、`文件库管理`、`智能体列表`、`Endpoint 管理` 和 `审计日志`。这几页基本覆盖了日常使用、复杂执行、资料沉淀、资源接入和后续回看。
