# 界面与功能说明

这一组页面展示的是 AgentSmith 中最重要的工作界面。你会看到成员如何在项目里发起对话、创建任务、整理文件、接入智能体、管理模型资源，以及在需要时查看消耗和审计记录。换句话说，这一组内容不是单纯带你认页面，而是帮助你理解团队在系统里究竟怎样完成工作。

如果你想先建立对产品的整体直觉，最值得先看的通常是 `项目总览`、`Chat 多轮会话`、`Notebook 任务详情`、`文件库管理`、`智能体列表`、`Endpoint 管理` 和 `审计日志`。这几页基本覆盖了团队最常发生的几件事：开始交流、推进复杂任务、保存资料和结果、接入执行能力、管理模型资源，以及回看关键变化。
