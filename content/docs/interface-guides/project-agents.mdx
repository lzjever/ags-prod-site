# 智能体列表

<SiteSection title="界面概览" description="智能体列表页面用于查看可用智能体及其当前状态。">

智能体列表页用于集中查看当前项目中可用的智能体。用户可以直接看到智能体名称、在线状态、运行模式和交互方式，并据此判断某个智能体是否已经可用，适合用于什么类型的工作。

对于需要让多位成员共享执行能力的场景，这种集中展示方式会比依赖个人环境或零散说明更稳定。

</SiteSection>

<SiteSection title="主要价值" description="智能体列表让项目中的执行能力清晰可见，便于选择和管理。">

智能体的可用性和基本特征会在这里被清楚展示。成员不需要额外查找说明，就可以知道哪些智能体已经接入、当前状态是否正常，以及后续应从哪里继续配置或使用。这会让团队在共享智能体能力时更加清晰和可控。

</SiteSection>

<SiteSection title="相关功能" description="智能体列表与接入配置、任务执行和资源治理保持关联。">

从这里可以继续进入创建智能体、查看连接信息和配置服务密钥等页面；在实际使用时，智能体又会参与 Chat 和 Notebook 中的执行过程。它与 Endpoint、Credentials 和 Audit 等页面也保持直接关联。

</SiteSection>
