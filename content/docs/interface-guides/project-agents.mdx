<SiteSection title="你会在这里做什么" description="这里用于查看智能体清单、判断当前可用状态，并继续进入连接信息或任务使用。">

1. 查看当前项目下的智能体清单。
2. 判断智能体适用于聊天、Notebook 或双模式使用。
3. 继续打开连接信息或相关配置。

</SiteSection>

<SiteSection title="关键看点" description="这张页面最适合说明智能体能力如何被纳入项目级正式管理。">

1. 智能体不再只是个人脚本，而是可被发现、调用与治理的项目对象。
2. 页面直接展示模式、在线状态和交互方式，方便团队判断如何使用。
3. 它把“接入智能体”和“使用智能体”之间的关系变得可见且可管理。

</SiteSection>

<SiteSection title="关联模块" description="智能体列表连接接入、执行和治理三条主线。">

1. 向下连接创建智能体、连接信息和服务密钥等配置动作。
2. 向使用面连接 Chat 与 Notebook 中的实际执行能力。
3. 向治理面连接 Endpoint、Credentials 和 Audit 的资源与回看逻辑。

</SiteSection>
