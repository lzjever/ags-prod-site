<SiteSection title="你会在这里做什么" description="这里承接项目中的多轮对话、线程切换与结果查看。">

1. 展示会话线程与当前上下文切换。
2. 说明结果优先的对话式 AI 体验。
3. 用于承接“快速开始”和“低学习成本”的产品价值。

</SiteSection>

<SiteSection title="关键看点" description="Chat 页最适合用来说明 AgentSmith 如何降低团队使用 AI 的门槛。">

1. 多轮线程与当前对话在同一界面中连续展开，适合日常快速使用。
2. 用户不需要先理解复杂执行协议，也能直接获得项目级 AI 能力。
3. 对话结果仍然受到同一项目资源和治理规则约束。

</SiteSection>

<SiteSection title="关联模块" description="Chat 看似轻量，但它背后连接的是完整的项目能力体系。">

1. 调用的执行对象来自 Endpoint 或 Agent。
2. 输入与后续结果可与 Files、Notebook 继续衔接。
3. 使用行为继续进入 Usage 与 Audit 的治理范围。

</SiteSection>
