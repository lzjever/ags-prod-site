# 凭据管理

<StatusBadge state="已实现" />

  preset="project-credentials-guide"

<SiteSection title="界面概览" description="凭据管理页面用于查看和维护项目可用凭据。">

凭据管理页用于集中维护项目可用的凭据。用户可以在这里查看凭据名称、类型和轮换状态，并继续执行新增、替换或清理等操作。

</SiteSection>

<SiteSection title="主要价值" description="凭据管理让敏感连接信息能够以统一方式被维护和复用。">

敏感连接信息可以在这里被统一维护和复用。这样既方便模型和服务接入，也有助于后续轮换、审计和持续管理。

</SiteSection>

<SiteSection title="相关功能" description="凭据管理与模型资源接入、智能体连接和审计记录保持关联。">

凭据会继续被 Endpoint 和智能体配置引用，而相关变化也会进入审计记录。它是模型接入和外部服务连接的重要基础。

</SiteSection>
