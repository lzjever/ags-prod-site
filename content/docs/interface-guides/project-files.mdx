<SiteSection title="你会在这里做什么" description="这里用于查看项目文件库、目录内容与本地挂载信息。">

1. 解释项目文件库不是一次性附件区。
2. 说明本地挂载与平台内工作流的关系。
3. 说明 artifacts、inputs 与 files 的双向连接。

</SiteSection>

<SiteSection title="关键看点" description="Files 页最适合用来说明 AgentSmith 如何把文件从附件区升级成项目资产层。">

1. 文件库、目录浏览与详情面板在同一界面里形成稳定的工作台。
2. 文件既可以由成员上传，也可以承接任务执行生成的产物。
3. 本地挂载让 Web 端与本地工作目录形成连续协作关系。

</SiteSection>

<SiteSection title="关联模块" description="Files 与几乎所有使用面和运行链路都有直接关系。">

1. 为 Chat 和 Notebook 提供上下文输入来源。
2. 承接智能体任务生成的 artifacts 与长期资料。
3. 与挂载说明、删除限制、文件库创建等动作形成完整管理链路。

</SiteSection>
