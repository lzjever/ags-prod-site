<SiteSection title="你会在这里做什么" description="这里承接任务执行过程、运行状态与产物查看。">

1. 统一展示任务、过程、trace 和产物。
2. 让通用智能体执行从终端工作流变成可回看的项目内对象。
3. 是理解智能体执行方式与结果沉淀的关键界面。

</SiteSection>

<SiteSection title="关键看点" description="这张页面是整站里最适合体现 AgentSmith 差异化的一张执行界面。">

1. 任务消息、运行状态、trace 与产物都在同一界面中统一出现。
2. 智能体执行从“终端过程”变成“项目对象”，适合回看、复用与协作。
3. 这也是解释为什么 AgentSmith 不只是一个 Agent UI 的核心界面之一。

</SiteSection>

<SiteSection title="关联模块" description="任务详情页把输入、执行、产物与治理结果真正串联了起来。">

1. 输入来源可以来自 Files、URL 与历史 artifacts。
2. 执行能力来自 Agents 与 Endpoints。
3. 结果继续沉淀为文件资产，并进入 Audit 与 Usage 的观察范围。

</SiteSection>
