# 项目设置

<StatusBadge state="已实现" />

  preset="project-settings-guide"

<SiteSection title="界面概览" description="项目设置页面用于维护项目基础信息和访问规则。">

项目设置页用于维护项目的基础信息和访问规则。名称、描述、可见性和加入策略都会在这里被统一管理，并持续影响项目后续的成员协作和访问范围。

</SiteSection>

<SiteSection title="主要价值" description="项目设置帮助用户维护项目的长期边界和基础信息。">

项目边界可以在这里保持稳定且可持续维护。基础信息和访问规则集中更新后，后续的成员、资源和审计记录都会围绕同一项目定义继续运转。

</SiteSection>

<SiteSection title="相关功能" description="项目设置与成员治理、资源管理和审计记录保持关联。">

项目设置会直接影响成员访问范围，相关变更也会进入 Audit 留下记录，因此它始终与成员治理、资源管理和审计记录保持关联。

</SiteSection>
