# 用量与限额

<SiteSection title="界面概览" description="用量页面用于查看资源消耗、剩余额度和趋势变化。">

Usage 页面用于查看资源消耗、剩余额度和趋势变化。用户可以按 Endpoint 查看不同模型资源的使用情况，并据此判断当前资源状态是否正常。

</SiteSection>

<SiteSection title="主要价值" description="用量页面帮助用户及时理解资源状态并做出调整。">

资源状态会在这里被直接呈现出来，便于团队及时发现接近限额或消耗异常的情况。管理员也可以根据这些信息进一步调整策略或预算安排。

</SiteSection>

<SiteSection title="相关功能" description="用量页面与资源策略、审计记录和模型资源共同构成治理链路。">

Usage 观察的是 Endpoint，限制条件来自资源策略，而异常波动和关键变化又可以继续在 Audit 中追溯。它与资源管理和审计记录共同构成了一条完整的治理链路。

</SiteSection>
