# 用量与限额

<StatusBadge state="已实现" />

<ScreenshotFrame
  preset="project-usage-guide"

<SiteSection title="你会在这里做什么" description="这里用于查看剩余额度、支出与使用趋势。">

1. 选择目标 endpoint。
2. 查看限额卡片与趋势变化。
3. 与 Audit / Resource Policy 配合进行治理调整。

</SiteSection>

<SiteSection title="关键看点" description="用量页最适合说明治理不仅是限制，还包括对资源消耗的持续理解。">

1. 剩余额度、支出与趋势让成员知道当前资源使用情况。
2. 用户可以按 Endpoint 理解不同模型资源的消耗差异。
3. 这让资源治理从后台配置走向日常使用感知。

</SiteSection>

<SiteSection title="关联模块" description="Usage 是资源治理链路中最贴近日常使用体验的一层。">

1. 使用对象来自 Endpoint。
2. 限制条件来自 Resource Policy。
3. 异常波动和关键变化继续可以在 Audit 中被追溯。

</SiteSection>
