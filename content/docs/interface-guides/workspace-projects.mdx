# 工作区项目列表

<StatusBadge state="已实现" />

<SiteSection title="你会在这里做什么" description="这里用于浏览项目、搜索项目并创建新项目。">

1. 承接项目列表与搜索。
2. 对具备权限的成员开放创建项目功能。

</SiteSection>

<SiteSection title="关键看点" description="工作区项目列表最适合说明团队如何在同一工作区下组织多个项目。">

1. 项目列表、搜索与新建动作在同一界面中完成。
2. 这让 Workspace 与 Project 的层级关系在用户视角下非常清晰。
3. 页面承担了团队进入不同项目主线的实际分发作用。

</SiteSection>
