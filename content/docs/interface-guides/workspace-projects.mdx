# 工作区项目列表

<StatusBadge state="已实现" />

<SiteSection title="界面概览" description="工作区项目列表用于浏览、搜索和创建项目。">

工作区项目列表页用于浏览当前工作区中的项目、搜索已有项目，并创建新的项目。对于需要同时维护多个项目的团队来说，这一页提供了统一而清晰的管理入口。

</SiteSection>

<SiteSection title="主要价值" description="工作区项目列表帮助团队在同一空间下管理多个项目。">

同一工作区中的多个项目会在这里保持一致的组织方式。用户可以从这里快速定位已有项目，也可以在相同规则下创建新的项目，便于团队持续扩展和维护工作内容。

</SiteSection>
