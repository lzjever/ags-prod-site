# 工作区选择

<SiteSection title="界面概览" description="工作区选择页面用于进入可访问的工作区。">

工作区选择页用于帮助用户进入自己有权限访问的工作区。对于拥有多个业务空间或多个组织单元的场景，这一步能够先确定当前所处的工作范围，再继续进入后续项目。

</SiteSection>

<SiteSection title="主要价值" description="工作区选择页面帮助用户先确认业务空间，再继续进入项目。">

工作区边界会在所有操作开始前先被确认。用户先选择所在工作区，之后看到的项目列表、成员关系和资源范围都会与这一选择保持一致。

</SiteSection>
