# 工作区选择

<SiteSection title="你会在这里做什么" description="这里用于选择工作区，并进入对应的业务上下文。">

1. 展示可访问的 workspace 列表。
2. 确认接下来要进入的业务空间。

</SiteSection>

<SiteSection title="关键看点" description="工作区选择页最适合说明系统级入口与业务级入口之间的层级关系。">

1. 用户先确认自己要进入哪个业务空间，再继续项目操作。
2. 这体现了 Workspace 作为租户边界的存在，而不是简单分组。
3. 它为后续项目级使用和治理提供了明确的上层上下文。

</SiteSection>
