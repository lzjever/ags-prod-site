# 内容基线

当前站点内容并不是凭空撰写，而是建立在几类已经同步的资料之上: `references/prd-v1/`、`references/product-docs/latest/`，以及 `../agentsmith` 仓库中的代码、测试、合同与开发文档。也正因为内容持续锚定这些来源，站点才能同时兼顾产品表达和事实准确性。

<SiteSection title="状态表达" description="全站统一使用以下三种状态。">

<Cards>
  <Card title="已实现" description="已有真实页面、稳定功能链路或可验证实现依据。" />
  <Card title="部分实现" description="主链路存在，但后台闭环或基础设施仍未完全收口。" />
  <Card title="规划中" description="已有明确方向，但当前尚无完整实现依据。" />
</Cards>

</SiteSection>

<SiteSection title="当前界面资料" description="界面图片与说明默认使用当前同步版本。">
