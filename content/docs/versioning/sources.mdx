# 资料来源

当前内容主要依赖三类资料来源: `references/prd-v1/`、`references/product-docs/latest/`，以及 `../agentsmith` 仓库中的代码、测试、合同与开发文档。站点中的产品描述、界面说明和状态判断都会尽量以这些来源为准，而不是脱离实现现状单独书写。

<SiteSection title="更新方式" description="当界面资料或产品资料变化时，先同步最新参考内容。">

```bash
./scripts/sync-reference-sources.sh
```

</SiteSection>

<SiteSection title="为什么需要这些资料" description="保持资料一致，才能准确反映产品能力与当前状态。">

这些资料之所以重要，是因为它们帮助站点始终贴近真实产品。只有持续对齐资料来源，才能避免内容与主产品脱节，避免把规划能力写成已实现，也才能让界面图片、产品定义与当前实现情况始终保持一致。

</SiteSection>
