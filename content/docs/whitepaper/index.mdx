# AgentSmith 产品摘要

AgentSmith 是 MBOS 面向企业场景打造的多租户 AI 控制平面，同时也是通用智能体面向团队落地的运行环境。

它要解决的不是“如何再接一个模型 API”，而是企业在真实落地 AI 时最常见的结构性问题：

1. 多团队如何在一个可治理环境中使用 AI。
2. Chat、Notebook、Files、Agents 如何共享同一套资源与约束。
3. 谁在使用什么资源、消耗多少、哪里出现异常，如何追溯。
4. 通用智能体如何摆脱“本地命令行 + 临时目录 + 手工配置”的高门槛运行方式。
