# 定位与范围边界

AgentSmith 的正式定位是：

1. MBOS 的企业级多租户控制平面。
2. 企业级 AI 智能体使用与管理平台。
3. 项目级 AI 资源治理与审计平台。
4. 通用智能体的易用且安全的运行环境与统一用户接口。

<SiteSection
  title="在范围内"
  description="当前站点应持续围绕真实产品边界组织叙事，不把实现细节和未来想象混在一起。"
>

1. 系统管理与工作区生命周期。
2. 工作区与项目治理。
3. Chat、Notebook、Files、Agents 等使用面。
4. Endpoints、Policy、Credentials、Members、Usage、Audit、Settings 等治理面。
5. Agent 执行链路、任务工作目录、凭据文件、skills 与文件沉淀。

</SiteSection>

<SiteSection
  title="不在范围内"

1. DevOps 发布编排平台。
2. 组织级总控治理叙事。
3. 独立于项目主线之外的第三产品面。
4. 与当前主线无关的文件级策略扩张。
